# Meter - Feet Converter

## Usage

```console
$ go build go-hello/ch02/ex02/fm
$ ./fm 32
32m = 104.9856ft, 32ft = 9.75371860521824m
$ ./fm 32 -1045
32m = 104.9856ft, 32ft = 9.75371860521824m
-1045m = -3428.436ft, -1045ft = -318.51987320165813m
$ ./fm
32
32m = 104.9856ft, 32ft = 9.75371860521824m
-1045 
-1045m = -3428.436ft, -1045ft = -318.51987320165813m
```
